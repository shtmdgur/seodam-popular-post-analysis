# 서담 인기글 분석 및 LLM 기반 인기글 생성

> 서강대학교 커뮤니티 '서담'의 인기글 패턴을 분석하고, LLM을 파인튜닝하여 인기글 스타일의 게시글을 생성하는 프로젝트

## 📌 프로젝트 개요

서강대학교 커뮤니티 '서담'의 인기글 분석 프로젝트입니다.

5개 게시판(인기글, 자유게시판, 연애상담소, 익게1, 익게2)에서 약 **6만 건의 게시글**을 수집하고, BERTopic 기반 토픽 모델링으로 게시판별 주제 구조와 인기글 패턴을 정량화하였습니다.

분석 결과를 Llama-3-8B 모델에 QLoRA 파인튜닝으로 학습시켜 인기글 스타일의 게시글을 생성하고, **실제 커뮤니티에 게시하여 인기글로 등극하는 성과**를 달성함으로써, **분석 → 생성 → 검증의 Closed-loop 파이프라인**을 구현하였습니다.

## 🛠 기술 스택

|------|------|
